第1回 WWW と HTML

HTMLの初歩的なページ編集

<p>や<strong>などのページを作る際の基本となる文書を学んだ。 HTML文書は基本的にタグで囲んでその中に必要に応じて入れ子を入れるなどがある。 そして最も重要なのが終わりのタグには/（スラッシュ）をつけなければいけないことである。

HTML文書の構成は以下のようなテンプレートからなる。 

<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>？？？</title>
  </head>
  <body>
    <p>？？？</p>
  </body>
</html>

<title></title> と <p></p> の間に文字を入れたり、 <p></p> の中に <strong></strong> を入れることができる。

加えて、同じファイル内にある別のHTML文書をハイパーリンクとして載せることもできるようになった。 ファイル外のものを指定する際には手順が異なるため、注意が必要であると感じた。
